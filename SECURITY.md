# Security Policy

This repository contains only trained model weights and documentation. It does not include any production code and therefore has minimal security surface area. Nevertheless, we welcome responsible disclosure of any issues related to the content herein.

## Supported Versions

This project provides documentation and model files without version‑specific code. There are no supported software versions.

## Reporting a Vulnerability

If you believe you have found a security vulnerability in this repository, please email the project maintainers or open a private security advisory on GitHub. Do not disclose details publicly until we have had a chance to investigate.
